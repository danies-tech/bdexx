# VeriMesh - Web3 Wallet Services Platform

## Overview
VeriMesh is a Web3 application that provides wallet connection and validation services, similar to Moralis. The application features a modern landing page with Web3/crypto aesthetics and real wallet integration capabilities.

## Core Features

### Landing Page
- Modern, responsive design with Web3/crypto aesthetic
- Dark theme with bold typography and smooth animations
- Hero section with prominent call-to-action buttons
- Professional layout similar to Moralis platform

### Wallet Integration
- Prominent "Wallet Connect" button on homepage
- Prominent "Wallet Validation" button on homepage
- Real wallet connection functionality using MetaMask integration
- Wallet validation features to verify connected wallets
- Display connected wallet address and connection status

### Services Section
- Dedicated section explaining VeriMesh's crypto problem-solving services
- Short descriptions of key services including:
  - Wallet security solutions
  - Connectivity support
  - Validation services
  - Other Web3 infrastructure services

### User Interface
- Smooth animations and transitions
- Responsive design for desktop and mobile
- Clean, modern Web3-themed styling
- Bold typography consistent with crypto/blockchain aesthetics

## Technical Requirements

### Frontend
- Implement MetaMask wallet connection
- Handle wallet connection states (connected, disconnected, error)
- Display wallet address when connected
- Responsive design with smooth animations
- Modern Web3 UI components

### Backend
- Store basic analytics data about wallet connections
- Track validation requests
- No user authentication required
- Simple data persistence for usage metrics

## User Flow
1. User visits VeriMesh landing page
2. User reads about services in the explanation section
3. User clicks "Wallet Connect" to connect their MetaMask wallet
4. System displays connection status and wallet address
5. User can use "Wallet Validation" to verify their connected wallet
6. User can disconnect wallet if needed
